## 🛠️ 実装方針

### データ構造の拡張

#### 1. `data/members.json` に追加するフィールド
```json
{
  "name": "山田太郎",
  "email": "yamada@example.com",
  "company": "株式会社サンプル",
  "industry": "経営コンサル",
  "photo": "uploads/members/yamada.jpg",
  "birthday": "1980-05-15",
  "joined_date": "2024-10-31",
  "team": "A",
  "role": "メンバー"
}
```

#### 2. アンケートCSVに追加するフィールド
- 現状維持（既存フィールドを活用）

#### 3. 新規設定ファイル `data/slide_config.json`
```json
{
  "president_message": "本日も宜しくお願いします",
  "secretary_message": "会費の納入をお願いします",
  "director_message": "次回の研修についてお知らせします",
  "visitor_hosts": ["野口浩乃", "吉田惇哉", "渡邊真造"],
  "speaker_rotation": [
    {
      "date": "11/7",
      "presenter": "吉岡恭介",
      "target": "高齢者の方々に関わる仕事"
    }
  ]
}
```

### 段階的実装ステップ

#### Phase A: データ準備（まず最初）
1. `data/members.json` に写真パス、誕生日、入会日を追加
2. `data/slide_config.json` を作成
3. メンバー写真を `uploads/members/` にアップロード

#### Phase B: スライド生成ロジック拡張
1. `assets/js/svg-slide-generator.js` を段階的に拡張
2. 各フェーズごとに1セクションずつ実装
3. 実装後、必ずテスト

#### Phase C: 管理画面の拡張
1. `slide_config.json` の編集画面を追加
2. メンバー写真アップロード機能
3. スピーカーローテーション管理画面
