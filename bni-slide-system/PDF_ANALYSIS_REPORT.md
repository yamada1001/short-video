## 4. 修正計画

### 結論

**本番PDFの309ページのうち、約70%（約200ページ）は固定スライド**であり、**BNI Slide Systemで動的に生成すべきスライドは約30%（約100ページ）のみ**です。

### 推奨アプローチ

#### Option 1: チャプター固定スライドを別途管理（推奨）

BNI Slide Systemは**動的データのみ**を生成し、チャプター固定スライドは別途PowerPointで管理・挿入する。

**BNI Slide Systemで生成すべきスライド**:
1. ローテーション（p9-13, p199-203, p297-301）
2. ビジター紹介リスト（p19, p37, p169-223）
3. 代理出席（p22-24）※新規実装
4. 新入会メンバー（p25-27, p99-104）※新規実装
5. メンバープレゼン（p107-166）
6. ビジター一言感想（p212-224）※新規実装
7. バイスプレジデント進捗報告（p187-190）※新規実装

**チャプター独自で管理すべき固定スライド**:
- Opening（p2-8）
- 教育関連（p14-16）
- 委員会紹介（p42-58）
- BNI教育（p61-72）
- ネットワーキング学習（p75-86）
- チャンピオン表示（p89-96）
- ビジターオリエンテーション（p236-282）
- 終礼（p284-308）

#### Option 2: svg-slide-generator.jsを大幅に拡張

すべてのスライドをBNI Slide Systemで生成できるように機能拡張する。

**必要な実装**:
1. 固定スライドテンプレート機能
2. チャプター設定による固定スライドカスタマイズ
3. 委員会情報データベース
4. BNI教育スライドテンプレート
5. ビジターオリエンテーションスライドテンプレート
6. 終礼スライドテンプレート

**推定作業量**: 非常に大（100時間以上）

### 最小限の修正（現在のシステムを本番に近づける）

現在のsvg-slide-generator.jsのgenerateSVGSlides()関数の順序を、本番PDFの**動的スライド部分のみ**に合わせて修正：

\`\`\`javascript
async function generateSVGSlides(data, stats, slideDate = '', ...) {
  let slides = '';

  // Phase 1: ローテーション（本番PDFではp9-13で最初に表示）
  if (rotationData) {
    slides += generateRotationSlides(rotationData);
  }

  // Phase 2: ビジター紹介リスト（p19, p37, etc）
  if (visitorIntroductions && visitorIntroductions.length > 0) {
    slides += generateVisitorIntroductionListSlides(visitorIntroductions);
  }

  // Phase 3: 代理出席（p22-24）※新規実装必要
  if (proxyAttendance && proxyAttendance.length > 0) {
    slides += generateProxyAttendanceSlides(proxyAttendance);
  }

  // Phase 4: 新入会メンバー（p25-27）※新規実装必要
  if (newMembers && newMembers.length > 0) {
    slides += generateNewMemberSlides(newMembers);
  }

  // Phase 5: メンバープレゼン（p107-166）
  if (slideConfig && slideConfig.members && slideConfig.members.length > 0) {
    slides += generateMemberPitchSlides(slideConfig.members);
  }

  // Phase 6: ビジター自己紹介（p169-180、ベル通知付き）
  if (visitorIntroductions && visitorIntroductions.length > 0) {
    slides += generateVisitorSelfIntroductionSlidesWithBell(visitorIntroductions);
  }

  // Phase 7: ビジター一言感想（p212-224）※新規実装必要
  if (visitorFeedback && visitorFeedback.length > 0) {
    slides += generateVisitorFeedbackSlides(visitorFeedback);
  }

  // Phase 8: VP進捗報告（p187-190）※新規実装必要
  if (vpProgressData) {
    slides += generateVPProgressSlides(vpProgressData);
  }

  slideContainer.innerHTML = slides;
}
\`\`\`

### 新規実装が必要な関数

1. `generateProxyAttendanceSlides(proxyAttendance)` - 代理出席スライド
2. `generateNewMemberSlides(newMembers)` - 新入会メンバースライド
3. `generateVisitorSelfIntroductionSlidesWithBell(visitors)` - ベル通知付きビジター自己紹介
4. `generateVisitorFeedbackSlides(feedback)` - ビジター一言感想
5. `generateVPProgressSlides(progressData)` - VP進捗報告（歴史、累計リファーラル、売上）
