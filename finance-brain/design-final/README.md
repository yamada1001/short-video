# Design Final - デザイン完成版ディレクトリ

このディレクトリは、モックアップを基に実際のデザイン・コーディングを進めるための作業用ディレクトリです。

## 📂 ディレクトリ構成

```
design-final/
├── index.html              # トップページ（デザイン完成版）
├── about/                  # 会社紹介
├── services/               # サービスページ
│   ├── personal/          # 個人向け
│   └── corporate/         # 法人向け
├── why-us/                # 選ばれる理由
├── voice/                 # お客様の声
├── staff/                 # スタッフ紹介
├── company/               # 会社情報
├── news/                  # お知らせ
├── faq/                   # よくあるご質問
├── contact/               # お問い合わせ
└── assets/                # 静的ファイル
    ├── css/
    ├── js/
    ├── images/
    └── fonts/
```

## 🎨 デザイントンマナ（トーン＆マナー）

### カラーパレット

**プライマリーカラー（メイン）:**
- `#5767bf` - 濃いブルー（現サイトから継承）
- 使用箇所: ヘッダー、メインナビゲーション、見出し、CTA背景

**セカンダリーカラー:**
- `#3a4a8f` - より深いネイビー（グラデーション用）
- `#a0b3e0` - 淡いブルー（背景・装飾用）

**アクセントカラー:**
- `#ff8c42` - オレンジ（CTA・強調用）
- 使用箇所: ボタン、お問い合わせリンク、重要な情報

**背景カラー:**
- `#ffffff` - 白（メイン背景）
- `#f5f7fa` - オフホワイト（セクション背景）
- `#fafbfc` - ライトグレー（交互背景）

**テキストカラー:**
- `#333333` - ダークグレー（本文テキスト）
- `#666666` - ミディアムグレー（補足テキスト）
- `#999999` - ライトグレー（キャプション）

### フォント

**日本語フォント:**
- `'Noto Sans JP', 'Hiragino Kaku Gothic ProN', 'Hiragino Sans', 'Yu Gothic', 'Meiryo', sans-serif`

**英数字フォント:**
- `'Roboto', 'Arial', sans-serif`

**フォントサイズ:**
- 本文: `16px` (1rem)
- 見出し1: `32px` (2rem)
- 見出し2: `28px` (1.75rem)
- 見出し3: `24px` (1.5rem)
- 見出し4: `20px` (1.25rem)
- 小テキスト: `14px` (0.875rem)

**行間:**
- 本文: `1.7`
- 見出し: `1.4`

### レスポンシブブレークポイント

### デザインスタイル

**1. ミニマル・モダン**
- 豊富な余白（ホワイトスペース）
- グリッドベースレイアウト
- シンプルなタイポグラフィ
- 控えめなアニメーション

**2. 信頼感の演出**
- 専門資格バッジの表示
- 実績数値の可視化
- お客様の声・事例紹介
- スタッフ顔写真の掲載

**3. 親しみやすさ**
- 柔らかいイラスト・アイコン
- 平易な言葉での説明
- FAQの充実
- LINE相談ボタンの目立つ配置

### UIコンポーネント

**ボタン:**
- プライマリーボタン: 背景 `#5767bf`, テキスト `#ffffff`, ホバー時 `#3a4a8f`
- セカンダリーボタン: 背景 `#ff8c42`, テキスト `#ffffff`, ホバー時 `#e67e22`
- アウトラインボタン: ボーダー `#5767bf`, テキスト `#5767bf`, ホバー時背景 `#5767bf`

**カード:**
- 背景: `#ffffff`
- ボーダー: `1px solid #e0e0e0`
- ボックスシャドウ: `0 2px 8px rgba(0, 0, 0, 0.05)`
- ホバー時: `0 4px 12px rgba(0, 0, 0, 0.1)`
- 角丸: `8px`

**セクション:**
- パディング: `80px 0`（PC）, `60px 0`（SP）
- 交互背景: `#f5f7fa` と `#ffffff`

### アニメーション

**トランジション:**
- 標準: `all 0.3s ease`
- ホバー: `transform 0.2s ease`

**使用可能なアニメーション:**
- フェードイン
- スライドイン（上から・下から・左から・右から）
- スケール（拡大・縮小）
