# Protea - キーワード一覧シート作成ツール

Google Apps Scriptで「依頼キーワード一覧」シートを自動作成するツールです。

## 機能

- ✅ 38件のキーワードデータを自動挿入
- ✅ 2025年11月・12月の集計表を自動作成
- ✅ ステータス「作成済」は緑背景で表示（条件付き書式）
- ✅ プロフェッショナルなデザイン（ヘッダー青背景・白文字）
- ✅ 列幅・行高さを自動調整

## セットアップ

### 1. Googleスプレッドシートを作成

1. https://sheets.google.com で新しいスプレッドシートを作成
2. 「拡張機能」→「Apps Script」を開く

### 2. GASコードをデプロイ

1. デフォルトの`コード.gs`を削除
2. `keyword_sheet_creator.gs`の内容を全てコピー＆ペースト
3. 💾 保存（Ctrl+S / Cmd+S）

### 3. シートを作成

1. スプレッドシートをリロード（F5 / Cmd+R）
2. メニュー「🔧 ツール」→「📋 キーワード一覧シートを作成」をクリック
3. 権限承認を求められたら承認
4. ✅ 完成！

## シート構成

### 列

| 列 | 項目 | 説明 |
|----|------|------|
| A | 番号 | 1-99 |
| B | キーワード | SEOキーワード |
| C | ステータス | 作成済・未着手など |
| D | 納品URL | AI記事生成用シートURL |
| E | 備考 | メモ欄 |
| F | 対象月 | 2025/11, 2025/12 |
| G | 森確認 | チェック欄 |
| H-K | 集計表 | 月別本数・単価・計 |

### 集計表

**11月分**（行2-3）
- 本数: 5件
- 単価: 500円
- 合計: ¥2,500

**12月分**（行5-6）
- 本数: 33件
- 単価: 600円
- 合計: ¥19,800

## デザイン

- **ヘッダー**: 青背景（#4A90E2）・白文字・太字
- **ステータス「作成済」**: 緑背景（#D4EDDA）
- **集計表**: 青・グレー・ライトブルーの配色
- **枠線**: 全セルに薄いグレー枠線

## カスタマイズ

### キーワードを追加する場合

`getKeywordData()` 関数の配列に行を追加：

```javascript
[39, '新しいキーワード', '未着手', '', '', '2025/12', '', '', '', '', '', '']
```

### 集計表を変更する場合

`createSummaryTable()` 関数の数値を変更：

```javascript
sheet.getRange(3, 10).setValue(5);  // 本数
sheet.getRange(3, 11).setValue(500);  // 単価
sheet.getRange(3, 12).setValue('¥2,500');  // 計
```
