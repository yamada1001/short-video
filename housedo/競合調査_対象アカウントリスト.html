調査項目（各アカウント共通）

1. コンテンツ内容

  • どんなテーマを扱っているか
  • 専門性 vs エンタメ性のバランス
  • シリーズ化・体系化されているか

2. トーン・雰囲気

  • 真面目 / 親しみやすい / プロフェッショナル / コミカル
  • 話し方・テンポ
  • BGM・効果音の使い方

3. 出演者・演出

  • 顔出し有無
  • 社長 / スタッフ / 専門家
  • キャラクター性（「税理士芸人」等）

4. 構成・編集

  • 動画の長さ（15秒 / 30秒 / 60秒 / 90秒）
  • テロップの使い方
  • サムネイルデザイン
  • 撮影場所（オフィス / 現場 / スタジオ）

5. CTA（行動喚起）

  • どう誘導しているか（DM / LINE / 問い合わせフォーム / イベント / ホワイトペーパー）
  • 中間CV設計（リスト獲得）の有無
  • プロフィールのリンク先

6. エンゲージメント（参考値）

  • いいね数・コメント数
  • 再生数
  • 保存数（取得できれば）
