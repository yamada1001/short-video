そのお悩み、放置すると危険です

資金繰りの悪化は、突然やってきます

「売上は伸びているから大丈夫」と思っていても、売掛金の回収サイトが長かったり、在庫が増えていたりすると、 あっという間に資金ショートに陥ります。黒字倒産の企業の約60%が、資金繰り管理の不備が原因と言われています。

実際に、年商5億円で営業利益率10%の優良企業が、急激な受注増加により運転資金が不足し、 資金ショートで倒産したケースもあります。「利益が出ているから安心」ではないのです。

事業承継対策は、10年前から始めるべきです

事業承継は「いつか考えよう」では手遅れになります。自社株の評価額が1億円の場合、 相続税は約3,000万円かかる可能性があります。これを後継者が現金で支払うのは現実的ではありません。

しかし、10年計画で生前贈与や株価対策を行えば、税負担を数百万円に抑えることも可能です。 「まだ若いから」「まだ元気だから」と先延ばしにすると、選択肢が大きく狭まってしまいます。

優秀な人材の流出は、企業の成長を止めます

「うちは中小企業だから退職金はなくても仕方ない」と諦めていませんか？ しかし、退職金制度のある企業とない企業では、離職率に約2倍の差があるというデータもあります。

せっかく育てた優秀な人材が「将来が不安だから」と退職してしまうのは、企業にとって大きな損失です。 少ない負担で始められる退職金制度もありますので、まずはご相談ください。

財務コンサルティング

こんな課題を解決します

  • • 売上は伸びているのに、手元にお金が残らない
  • • 決算書は税理士任せで、経営判断に活用できていない
  • • 銀行から融資を受けたいが、事業計画書の作り方が分からない
  • • 月次の資金繰り表がなく、いつも資金不足に悩まされている

なぜこの課題が発生するのか？

多くの中小企業では、「税理士に決算を任せているから大丈夫」と考えがちです。 しかし、税理士の主な役割は税務申告であり、経営コンサルティングではありません。 その結果、「利益は出ているのにお金がない」という事態に陥ります。 これは、売掛金の増加、在庫の増加、設備投資などで現金が減少しているからです。

ファイナンスブレーンの解決策

私たちは、財務諸表を単なる税務資料ではなく、経営判断のためのツールとして活用します。 キャッシュフロー分析を行い、「お金の流れ」を可視化。どこに問題があるのかを明確にします。

  • • 財務諸表の分析：損益計算書・貸借対照表・キャッシュフロー計算書を分析し、経営課題を抽出
  • • 資金繰り表の作成：今後6ヶ月〜1年の資金繰りを予測し、資金ショートを未然に防止
  • • 事業計画書の作成サポート：銀行融資に必要な説得力のある事業計画書を作成
  • • 銀行交渉のアドバイス：金融機関との交渉のポイント、必要資料の準備をサポート
  • • 経営改善計画の策定：売上向上・コスト削減の具体策を提案
  • • 定期的なモニタリング：月次・四半期ごとに進捗を確認し、PDCAを回す

解決後のメリット

  • • 資金繰りが安定し、支払い遅延の心配がなくなる
  • • 数字に基づいた経営判断ができるようになる
  • • 銀行からの融資が受けやすくなる
  • • 社員への給与還元や設備投資など、攻めの経営ができる

実際の改善事例

【製造業A社：年商3億円】

課題：売上は順調だが、資金繰りが厳しく、銀行借入の返済が重荷になっていた。

実施した対策：

  • • キャッシュフロー分析の結果、在庫過多が原因と判明
  • • 在庫圧縮計画を策定し、3ヶ月で在庫を30%削減
  • • 売掛金回収サイトを60日→45日に短縮交渉
  • • 余剰資金で借入金を一部繰上返済し、金利負担を軽減

結果：月次の手元資金が平均800万円増加。資金繰りの不安がなくなり、新規設備投資も実現。

退職金コンサルティング

こんな課題を解決します

  • • 優秀な人材が定着せず、3年以内の離職率が高い
  • • 退職金制度を導入したいが、資金的に厳しい
  • • 役員退職金をいくら準備すればいいか分からない
  • • 退職金規程はあるが、実際に支払えるか不安

なぜこの課題が発生するのか？

中小企業の約40%は退職金制度がないと言われています。「うちは中小企業だから」と諦めている経営者も多いですが、 退職金制度のある企業とない企業では、離職率に約2倍の差があります。 特に30代の優秀な人材は、将来を見据えて退職金制度のある企業を選ぶ傾向が強くなっています。

ファイナンスブレーンの解決策

退職金制度は、会社の規模や資金状況に応じて柔軟に設計できます。 少ない負担で始められる制度もありますので、まずは現状をお聞かせください。

  • • 退職金制度の設計：会社の規模・資金力に応じた最適な制度を設計
  • • 企業型確定拠出年金（企業型DC）の導入：月5,000円から始められ、全額損金算入可能
  • • 確定給付企業年金（DB）の活用：ある程度の規模がある企業向けの制度
  • • 役員退職金の算定：税務上適正な金額を算定し、計画的に準備
  • • 退職金規程の作成：社会保険労務士と連携し、法令に準拠した規程を作成
  • • 税制優遇措置の活用：中小企業退職金共済（中退共）など、国の制度を最大限活用

解決後のメリット

  • • 優秀な人材の定着率が向上し、採用コストが削減できる
  • • 従業員のモチベーションが上がり、生産性が向上する
  • • 税制優遇措置により、節税しながら退職金を積み立てられる
  • • 役員退職金を計画的に準備でき、将来の不安が解消される

実際の導入事例

【建設業B社：従業員15名】

課題：30代の優秀な職人が「将来が不安」と退職。採用しても3年以内に辞めてしまう悪循環。

実施した対策：

  • • 中小企業退職金共済（中退共）に加入し、従業員1人月5,000円から積立開始
  • • 企業型確定拠出年金を併用し、月15,000円の積立を実現
  • • 退職金規程を整備し、求人票に「退職金制度あり」と明記

結果：離職率が50%→15%に低下。「退職金があるから安心して働ける」と従業員満足度も向上。

事業承継対策

こんな課題を解決します

  • • 後継者はいるが、いつどう引き継げばいいか分からない
  • • 相続税がいくらかかるか不安で、夜も眠れない
  • • 自社株の評価額が高く、承継時の税負担が心配
  • • 後継者がおらず、M&Aも検討しているが、どこに相談すればいいか分からない

なぜこの課題が発生するのか？

中小企業庁の調査によると、事業承継の準備を始めてから完了まで、平均5〜10年かかると言われています。 しかし、多くの経営者は「まだ若いから」「まだ元気だから」と先延ばしにしてしまいます。 その結果、いざ承継しようとしたときには選択肢が限られてしまい、 多額の税金を支払わざるを得なくなるケースが後を絶ちません。

ファイナンスブレーンの解決策

事業承継は、計画的に進めることで税負担を大きく軽減できます。 私たちは、税理士・弁護士・M&A仲介会社などの専門家と連携し、最適な承継プランを設計します。

  • • 事業承継計画の策定：5年〜10年計画で、段階的に株式を移転するスケジュールを作成
  • • 後継者育成プラン：経営者として必要なスキル・知識を身につけるための育成計画
  • • 株式移転スキームの提案：生前贈与・遺言・信託など、最適な方法を提案
  • • 事業承継税制の活用：要件を満たせば、相続税・贈与税の納税猶予が受けられる制度を活用
  • • M&Aのご相談：後継者がいない場合、信頼できるM&A仲介会社をご紹介
  • • 相続税・贈与税対策：税理士と連携し、税負担を最小化する対策を実施

解決後のメリット

  • • 計画的に承継を進めることで、税負担を大幅に軽減できる
  • • 後継者が安心して経営を引き継げる環境が整う
  • • 従業員や取引先に不安を与えず、円滑に承継できる
  • • 引退後の生活資金も確保でき、安心してリタイアできる

実際の承継事例

【卸売業C社：年商5億円、自社株評価額1.2億円】

課題：長男に事業を承継したいが、自社株の評価額が高く、相続税が約3,600万円かかると試算された。

実施した対策：

  • • 10年計画で生前贈与を実施（暦年贈与の活用）
  • • 自社株の評価額を引き下げる対策を実施（後述の「自社株対策」参照）
  • • 事業承継税制を活用し、残りの株式は納税猶予を受ける

結果：税負担を約400万円に抑えることに成功。長男も安心して経営を引き継ぎ、会社は順調に成長。

自社株対策

こんな課題を解決します

  • • 自社株の評価額がいくらか分からない
  • • 税理士に「自社株の評価が高い」と言われたが、どうすればいいか分からない
  • • 株価を下げる方法があると聞いたが、具体的にどうすればいいか分からない
  • • 事業承継税制を使えば本当に税金がゼロになるのか不安

なぜこの課題が発生するのか？

自社株の評価額は、会社の業績が良いほど高くなります。これは喜ばしいことですが、 事業承継の際には、高額な相続税・贈与税という形で重い負担になります。 例えば、自社株の評価額が1億円の場合、相続税は約3,000万円。 後継者がこれを現金で支払うのは現実的ではありません。

ファイナンスブレーンの解決策

自社株対策は、早めに着手することで選択肢が広がります。 株価を適正に評価し、計画的に引き下げることで、税負担を大きく軽減できます。

  • • 自社株式の評価額算定：税理士と連携し、現在の評価額を正確に算定
  • • 株価引き下げ対策の提案：役員退職金の支給、不動産購入、持株会社の設立など、合法的な手法を提案
  • • 事業承継税制の活用サポート：要件を満たせば、相続税・贈与税の納税が猶予される制度を活用
  • • 持株会社スキームの活用：持株会社を設立し、株式を移転することで評価額を引き下げる
  • • 生前贈与による計画的な株式移転：毎年110万円の非課税枠を活用し、少しずつ株式を移転
  • • 種類株式の活用：議決権制限株式など、種類株式を活用した柔軟な承継スキーム

解決後のメリット

  • • 税負担を数百万円〜数千万円単位で軽減できる
  • • 後継者が現金負担なく、株式を承継できる
  • • 計画的に承継を進めることで、経営権の移転もスムーズに
  • • 事業承継後も会社経営が安定し、従業員・取引先に不安を与えない

実際の対策事例

【製造業D社：年商8億円、自社株評価額2億円】

課題：業績好調で自社株の評価額が2億円に。このまま承継すると相続税が約6,000万円かかると試算された。

実施した対策：

  • • 代表取締役の退職金を支給し、純資産を圧縮（株価が2億円→1.5億円に低下）
  • • 持株会社を設立し、株式を移転（評価額がさらに低下）
  • • 生前贈与を併用し、毎年110万円ずつ株式を移転
  • • 事業承継税制を活用し、残りの株式は納税猶予

結果：税負担を約500万円に抑えることに成功。6,000万円→500万円と、約5,500万円の節税に成功。

よくあるご質問

Q1. 相談料はかかりますか？

初回相談は無料です。御社の状況をお伺いし、最適なプランをご提案いたします。 ご相談だけで終わっても構いませんので、お気軽にお問い合わせください。

Q2. 顧問契約が必要ですか？

必須ではありません。スポット（単発）でのご相談も承っております。 ただし、財務改善や事業承継は継続的なサポートが効果的ですので、 顧問契約をご検討いただくケースが多いです。料金体系は御社の規模やニーズに応じて柔軟に対応いたします。

Q3. すでに税理士と契約していますが、大丈夫ですか？

はい、問題ございません。私たちは税理士ではなく、財務コンサルタントとして、 現在の税理士先生と連携しながらサポートいたします。 税務申告は税理士先生、財務改善・資金繰り・事業承継は私たちという形で、 役割分担をしながら進めることができます。

Q4. 小規模な会社でも対応してもらえますか？

もちろんです。従業員数名の小規模企業から、年商数億円の企業まで、 幅広くサポートしております。会社の規模に関わらず、経営者の皆様が抱える悩みは共通しています。 小規模企業だからこそ、財務面での専門的なサポートが必要だと考えています。

Q5. 事業承継対策は何歳から始めるべきですか？

50代から始めるのが理想的です。事業承継は5年〜10年計画で進めるのが一般的ですので、 「まだ早い」ということはありません。むしろ、早く始めるほど選択肢が増え、税負担を軽減できます。 60代、70代でも対策は可能ですが、選択肢が限られてしまうため、できるだけ早めのご相談をおすすめします。

Q6. M&Aの相談もできますか？

はい、可能です。後継者がいない場合、M&A（事業譲渡・売却）も有力な選択肢の一つです。 私たちは信頼できるM&A仲介会社と連携しており、御社に最適な譲渡先を見つけるサポートをいたします。 譲渡価格の算定、譲渡スキームの提案、税務対策など、トータルでサポートいたします。
