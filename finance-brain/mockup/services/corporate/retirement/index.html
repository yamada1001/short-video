  • 退職金コンサルティング

せっかく育てた優秀な人材が、
3年で辞めてしまう…

人材定着に悩む経営者様へ。退職金制度で、離職率を大幅に改善します。

こんなお悩みありませんか？

優秀な人材が定着しない

  • • せっかく育てた優秀な社員が、3年以内に辞めてしまう
  • • 「将来が不安だから」と言って退職してしまう
  • • 30代の中堅社員が次々と転職してしまう
  • • 離職率が高く、いつも人手不足に悩まされている

採用で不利になっている

  • • 求人を出しても、なかなか応募が来ない
  • • 「退職金制度がない」と言われて断られる
  • • 大企業に人材を取られてしまう
  • • 採用コストがかさんで、利益を圧迫している

退職金を準備したいが、資金的に厳しい

  • • 退職金制度を導入したいが、資金的に厳しい
  • • いくら積み立てればいいか分からない
  • • 中小企業だから退職金は無理だと諦めている
  • • 役員退職金をいくら準備すればいいか分からない

退職金規程が整備されていない

  • • 退職金規程がなく、いくら払えばいいか分からない
  • • 口約束だけで、トラブルになりそうで不安
  • • 退職金を支払えるか不安
  • • 税務上の問題がないか心配

その悩み、放置すると危険です

優秀な人材の流出は、企業の成長を止めます

「うちは中小企業だから退職金はなくても仕方ない」と諦めていませんか？ しかし、退職金制度のある企業とない企業では、離職率に約2倍の差があるというデータもあります。

特に30代の優秀な人材は、将来を見据えて退職金制度のある企業を選ぶ傾向が強くなっています。 「給料は同じくらいだけど、退職金がある方を選んだ」という声は珍しくありません。

せっかく時間とコストをかけて育てた優秀な人材が「将来が不安だから」と退職してしまうのは、 企業にとって大きな損失です。1人の社員を採用・育成するコストは、年収の2〜3倍とも言われています。

退職金がないと、ずっと採用で苦労し続けます

求人を出しても応募が来ない、面接で「退職金制度はありますか？」と聞かれて「ない」と答えると、 その場で辞退されてしまう…こんな経験はありませんか？

今や退職金制度は「あったら良い福利厚生」ではなく、「あって当たり前」になりつつあります。 特に優秀な人材ほど、将来を見据えて福利厚生を重視します。

退職金制度がないことで、採用で不利になり、人手不足に悩まされ、 既存社員に負担がかかり、さらに離職が増える…という悪循環に陥ります。

ファイナンスブレーンが解決します

退職金制度は、会社の規模や資金状況に応じて柔軟に設計できます。
少ない負担で始められる制度もありますので、まずは現状をお聞かせください。

専門資格保有者が対応

ファイナンシャルプランナー（FP）や投資診断士など、 金融分野の専門資格を持つスタッフが直接対応します。

社労士と連携

社会保険労務士と連携し、法令に準拠した退職金規程を作成。 労務トラブルを未然に防ぎます。

少ない負担で始められる

月5,000円から始められる制度もあります。 会社の規模や資金力に応じて、最適なプランをご提案します。

ファイナンスブレーンのサポート内容

1. 退職金制度の設計

会社の規模・資金力・業種に応じて、最適な退職金制度を設計します。 「月いくら積み立てればいいのか？」「どの制度を選ぶべきか？」を明確にします。

  • • 現状分析（従業員数、給与水準、退職者数の推移）
  • • 退職金シミュレーション（勤続年数別の退職金額）
  • • 必要積立額の算定
  • • 最適な制度の選定

2. 中小企業退職金共済（中退共）の導入サポート

月5,000円から始められ、国の助成金も受けられる中退共は、 中小企業にとって最も導入しやすい退職金制度です。全額損金算入でき、節税効果もあります。

  • • 中退共の仕組みと メリット・デメリットの説明
  • • 加入手続きのサポート
  • • 国の助成金申請サポート（新規加入で掛金の1/2を助成）
  • • 掛金の設定アドバイス

3. 企業型確定拠出年金（企業型DC）の導入サポート

従業員が自分で運用する企業型DCは、会社の負担を抑えながら、 従業員の資産形成をサポートできる制度です。全額損金算入でき、社会保険料の削減効果もあります。

  • • 企業型DCの仕組みとメリット・デメリットの説明
  • • 制度設計と導入手続きサポート
  • • 従業員向け投資教育のサポート
  • • 運営管理機関の選定サポート

4. 確定給付企業年金（DB）の活用

ある程度の規模がある企業向けの制度で、会社が運用リスクを負う代わりに、 従業員に確実な退職金を約束できる制度です。福利厚生の充実度をアピールできます。

  • • DBの仕組みとメリット・デメリットの説明
  • • 制度設計と導入手続きサポート
  • • 運用方針の策定サポート
  • • 年金数理人との連携

5. 役員退職金の算定と準備プラン

役員退職金は、税務上適正な金額でなければ、損金算入が認められません。 功績倍率法などを用いて、適正な金額を算定し、計画的に準備します。

  • • 役員退職金の適正額算定（功績倍率法）
  • • 積立計画の策定
  • • 生命保険を活用した準備方法
  • • 税務リスクのチェック

6. 退職金規程の作成サポート

社会保険労務士と連携し、法令に準拠した退職金規程を作成します。 口約束だけでは、退職時にトラブルになるリスクがあります。

  • • 退職金規程のドラフト作成
  • • 労働基準法に準拠した内容チェック
  • • 従業員への説明資料作成
  • • 就業規則への反映サポート

解決後のメリット

離職率が大幅に低下

退職金制度を導入することで、「将来が不安だから」という理由での退職が激減。 優秀な人材が定着し、採用・育成コストを大幅に削減できます。

採用が有利になる

求人票に「退職金制度あり」と記載できるようになり、応募数が増加。 優秀な人材を採用しやすくなります。

従業員のモチベーション向上

「この会社で長く働けば、退職金がもらえる」という安心感が、 従業員のモチベーションと生産性の向上につながります。

節税効果

退職金の掛金は全額損金算入でき、法人税の節税につながります。 国の助成金を受けられる制度もあります。

実際の導入事例

【建設業B社：従業員15名】

中退共導入で離職率が50%→15%に改善

課題

30代の優秀な職人が「将来が不安」と次々と退職。採用しても3年以内に辞めてしまう悪循環に陥っていた。 「退職金がないから」と面接で断られることも増えていた。

実施した対策

  • • 中小企業退職金共済（中退共）に加入し、従業員1人月5,000円から積立開始
  • • 企業型確定拠出年金を併用し、月15,000円の積立を実現
  • • 退職金規程を整備し、求人票に「退職金制度あり」と明記
  • • 国の助成金を活用し、導入コストを削減

結果

離職率が50%→15%に大幅低下。「退職金があるから安心して働ける」と従業員満足度も向上。 求人への応募数も1.5倍に増加し、優秀な人材を採用できるようになった。

よくあるご質問

Q1. 中小企業でも退職金制度は導入できますか？

はい、可能です。中小企業退職金共済（中退共）は、月5,000円から始められ、 国の助成金も受けられるため、中小企業でも導入しやすい制度です。 従業員数名の小規模企業でも導入実績があります。

Q2. 資金的に厳しいのですが、大丈夫ですか？

会社の資金状況に応じて、無理のない範囲で制度設計ができます。 月5,000円から始めて、業績に応じて徐々に増額することも可能です。 また、国の助成金を活用することで、導入コストを抑えることもできます。

Q3. どの制度を選べばいいですか？

会社の規模、資金力、従業員の年齢構成などによって、最適な制度は異なります。 中退共、企業型DC、DBなど、それぞれメリット・デメリットがありますので、 御社の状況をお伺いした上で、最適なプランをご提案いたします。

Q4. すでに働いている従業員も対象になりますか？

はい、制度導入時点で在籍している従業員も対象にできます。 ただし、過去の勤続年数をどう扱うかは、会社の判断次第です。 公平性を保ちながら、会社の負担も考慮した制度設計をご提案します。

Q5. 税務上の問題はありませんか？

適切に設計された退職金制度であれば、掛金は全額損金算入でき、税務上の問題はありません。 私たちは税理士と連携しながら、税務リスクのない制度設計をサポートいたします。
