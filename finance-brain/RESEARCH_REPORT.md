## 1. 現在のサイト分析

### 1.1 サイト構造

```
ホーム
├─ お知らせ
├─ 経営理念
├─ 事業概要
├─ 役員紹介
├─ 取扱保険会社
├─ 会社概要・アクセス
└─ お問い合わせ
```

### 1.2 現在のカラースキーム

- **メインカラー**: `#5767bf` （濃いブルー）
- **背景色**: `#ffffff` （白）
- **使用箇所**: ボーダー、デコレーション要素、box1クラスの角括弧デザイン

### 1.3 デザインの特徴

- レスポンシブ対応（750px以下でモバイル切り替え）
- テーブルレイアウト
- FontAwesomeアイコン使用
- スムーズなホバーエフェクト
- ページトップボタン（固定位置）
- Metasliderプラグイン使用

### 1.4 サービス内容

**事業内容**:
- 生命保険媒介業
- 損害保険代理業
- ファイナンシャルプランニング

**専門資格**:
- CFP®、AFP
- 1級FP技能士
- 宅建
- 投資診断士
- 相続診断士

**コンセプト**:
> 保険のみならず種々のご相談に対応できるきめ細かなサービス

## 2. 競合サイト調査

### 2.1 IFA法人GAIA（https://www.gaiainc.jp/）

**特徴**:
- フィーベース（残高連動報酬）モデル
- 預かり資産800億円
- JPX上場企業

**デザインの特徴**:
- モダンで上品なミニマリズム
- 深い青系をアクセントカラーに使用
- 大量の余白を活用
- 明朝体とサンセリフの組み合わせ
- 大型ビジュアル（親近感のある人物写真）
- グリッドベースの整然とした構成

**信頼感を出すデザイン要素**:
- 数値実績の可視化（契約更新率、家族契約率など）

- 実名顧客インタビュー
- JPX上場企業の証券コード表示

**デザインの特徴**:
- "Trust & Safety"をテーマライン表示
- 深い青色をメインカラーに使用
- オレンジをアクセントカラーに使用（問い合わせ・資料請求誘導）
- 堅実で信頼性重視のデザイン
- 証券投資専門サービスのイメージ

**レイアウト**:
- グローバルナビゲーション
- セクション化されたコンテンツエリア
- パートナー企業ロゴのグリッド表示
- フッターに規制機関リンク（金融庁、証券取引等監視委員会など）

### 2.3 THEO by お金のデザイン（https://theo.blue）

**特徴**:
- ロボアドバイザーによる資産運用
- 約30種類のETFに投資

**デザインの特徴**:
- 青/ティール系を主要ブランドカラーに使用
- モダンでミニマルな美学
- SVGイラスト・アイコンを多用
- フラットデザイン原則
- プロフェッショナルな画像とシンプルなグラフィックスの組み合わせ

**レイアウト**:
- レスポンシブデザイン（モバイル/デスクトップ別バージョン）
- モジュラーセクション、明確な視覚階層
- ヒーローセクションにアプリモックアップ画像
- カードベースレイアウト
- インフォグラフィックセクション

**UIコンポーネント**:
- セグメント化されたナビゲーションタブ
- カルーセル/スライダー要素
- アプリストアダウンロードボタン（iOS/Android）
- アイコンベースの機能説明

**印象**:
プロフェッショナルな金融サービスと親しみやすいモダンデザインのバランスで、投資の不安を軽減

## 3. ブルーカラーを使った金融系デザイントレンド（2024-2025）

### 3.1 カラートレンド

**基本配色**:
- **メインカラー**: 深い青（ネイビー、インディゴ）
- **ベースカラー**: 白、オフホワイト
- **アクセントカラー**: オレンジ、アンバー（CTA誘導用）

**配色パターン**:
- 青 × 白 の清潔感ある組み合わせ
- 青 × ネイビー の単色グラデーション
- 淡いパステルブルー × 白 の柔らかい印象

### 3.2 デザインスタイルトレンド

**2024-2025年の金融業界Webデザイントレンド**:
1. **アイソメトリック or 太線のイラスト**
2. **目を引く色使い**（赤、青、緑、黄色など）
3. **シンプルでキャッチーな美学**
4. **大きな数字を目立たせる表示**

**共通する要素**:
- クリーンでシンプルなレイアウト
- 豊富な余白（ホワイトスペース）
- アニメーション・イラストで親しみやすさ向上
- 数字の目立つ表示（金利、手数料など）
- 信頼構築要素（アドバイザー写真、顧客の声）

### 3.3 青色が持つ心理効果

**ブランディングにおける青色の効果**:
- **誠実さ・信頼性**の演出
- **爽快感・清潔感**
- **プロフェッショナル感**
- B2B業界、IT企業、金融機関で多用される

## 5. リニューアルに向けた提案方向性

### 5.1 カラースキーム案

**現在の `#5767bf` をベースに、以下の配色を検討**:

**案1: クラシック・トラスト**
- メインカラー: `#5767bf` （現在の濃いブルー）
- サブカラー: `#3a4a8f` （より深いネイビー）
- アクセントカラー: `#ff8c42` （オレンジ - CTA用）
- 背景: `#ffffff` / `#f8f9fa` （白/オフホワイト）

**案2: モダン・ソフト**
- メインカラー: `#5767bf` （現在の濃いブルー）
- サブカラー: `#a0b3e0` （淡いブルー）
- アクセントカラー: `#4a90e2` （明るいブルー）
- 背景: `#ffffff` / `#fafbfc` （白/ライトグレー）

**案3: プロフェッショナル・コントラスト**
- メインカラー: `#5767bf` （現在の濃いブルー）
- サブカラー: `#2c3e50` （ダークグレーブルー）
- アクセントカラー: `#e67e22` （オレンジ - 問い合わせ誘導）
- 背景: `#ffffff` / `#ecf0f1` （白/ライトグレー）

### 5.2 デザイン方向性

**推奨スタイル**:
1. **ミニマル・モダン**
   - 豊富な余白
   - グリッドベースレイアウト
   - シンプルなタイポグラフィ
   - 控えめなアニメーション

2. **信頼感の演出**
   - アドバイザー顔写真掲載
   - 実績数値の可視化
   - 顧客の声・事例紹介
   - 資格・許認可の明示

3. **親しみやすさ**
   - 柔らかいイラスト・アイコン使用
   - 平易な言葉での説明
   - FAQ充実
   - LINE公式アカウント等のSNS連携強化

### 5.3 サイト構造案

```
ホーム（リニューアル）
├─ ファイナンスブレーンとは（経営理念統合）
├─ サービス
│  ├─ 保険のご相談
│  ├─ 資産運用のご相談
│  ├─ ライフプランニング
│  └─ 相続・事業承継
├─ 選ばれる理由（新設）
├─ お客様の声（新設）
├─ よくあるご質問
├─ スタッフ紹介（役員紹介拡充）
├─ 会社概要
│  ├─ 取扱保険会社
│  └─ アクセス
├─ お知らせ
└─ お問い合わせ
```
