  • 事業承継対策

後継者はいるが、
相続税が3,000万円以上かかりそう…

事業承継に悩む経営者様へ。10年計画で税負担を大幅に軽減します。

こんなお悩みありませんか？

相続税の負担が心配

  • • 自社株の評価額が高く、相続税が3,000万円以上かかりそう
  • • 後継者が相続税を現金で払えるか不安
  • • 税理士に「対策が必要」と言われたが、何をすればいいか分からない
  • • 相続税で会社の資金繰りが悪化しないか心配

いつ、どう引き継げばいいか分からない

  • • 後継者はいるが、いつ引き継げばいいか分からない
  • • 株式をどうやって移転すればいいか分からない
  • • 生前贈与と相続、どちらが有利なのか分からない
  • • 事業承継計画の立て方が分からない

後継者の育成に不安

  • • 後継者がまだ若く、経営者としての経験が不足している
  • • 社員や取引先が後継者を認めてくれるか不安
  • • 自分がいなくなった後、会社が続くか心配
  • • 引退後も口を出してしまいそう

後継者がいない

  • • 子どもが会社を継ぐ意思がない
  • • 従業員に継がせたいが、資金がない
  • • M&Aを検討しているが、どこに相談すればいいか分からない
  • • 廃業すると、従業員や取引先に迷惑がかかる

その悩み、放置すると危険です

事業承継対策は、10年前から始めるべきです

中小企業庁の調査によると、事業承継の準備を始めてから完了まで、平均5〜10年かかると言われています。 しかし、多くの経営者は「まだ若いから」「まだ元気だから」と先延ばしにしてしまいます。

その結果、いざ承継しようとしたときには選択肢が限られてしまい、 多額の相続税を支払わざるを得なくなるケースが後を絶ちません。

例えば、自社株の評価額が1億円の場合、相続税は約3,000万円。 これを後継者が現金で支払うのは現実的ではありません。 しかし、10年計画で生前贈与や株価対策を行えば、税負担を数百万円に抑えることも可能です。

突然の相続は、会社を危機に陥れます

「まだ元気だから大丈夫」と思っていても、人生は何が起こるか分かりません。 準備をせずに突然相続が発生すると、会社が大混乱に陥ります。

株式の所有権が不明確、経営権が分散、相続税を払うために会社の資金を使う、 取引先や金融機関の信用が低下…こうした問題が一気に噴出します。

実際に、創業者が急逝し、事業承継の準備をしていなかったため、 相続人同士で株式の取り合いになり、会社が分裂してしまったケースもあります。 「まだ早い」ではなく、「今から準備」が正解です。

ファイナンスブレーンが解決します

事業承継は、計画的に進めることで税負担を大きく軽減できます。
税理士・弁護士・M&A仲介会社などの専門家と連携し、最適な承継プランを設計します。

専門家ネットワーク

税理士、弁護士、M&A仲介会社など、各分野の専門家と連携。 ワンストップで事業承継をサポートします。

19年の実績

2006年の創業以来、多数の中小企業の事業承継をサポート。 業種を問わず幅広い経験とノウハウがあります。

長期計画で税負担軽減

5年〜10年計画で段階的に株式を移転することで、 税負担を数千万円単位で軽減できます。

ファイナンスブレーンのサポート内容

1. 事業承継計画の策定

5年〜10年計画で、段階的に株式を移転するスケジュールを作成します。 「いつ、何を、どうするのか」を明確にし、計画的に承継を進めます。

  • • 現状分析（株式所有状況、後継者の状況、財務状況）
  • • 承継スケジュールの策定（5年〜10年計画）
  • • 年次アクションプランの作成
  • • 進捗管理とフォローアップ

2. 自社株の評価額算定

税理士と連携し、現在の自社株の評価額を正確に算定します。 「相続税がいくらかかるのか？」を明確にすることが、対策の第一歩です。

  • • 自社株の評価額算定（類似業種比準価額方式、純資産価額方式）
  • • 相続税・贈与税のシミュレーション
  • • 評価額を引き下げる方法の提案
  • • 定期的な再評価

3. 株式移転スキームの提案

生前贈与、遺言、信託など、最適な株式移転方法を提案します。 「生前贈与と相続、どちらが有利なのか？」を具体的な数字で比較します。

  • • 生前贈与のメリット・デメリット
  • • 暦年贈与の活用（年110万円まで非課税）
  • • 相続時精算課税制度の活用
  • • 遺言・信託の活用
  • • 種類株式の活用（議決権制限株式など）

4. 事業承継税制の活用サポート

要件を満たせば、相続税・贈与税の納税が猶予される事業承継税制を活用します。 うまく活用すれば、税負担を大幅に軽減できます。

  • • 事業承継税制の要件確認
  • • 特例承継計画の作成サポート
  • • 認定申請のサポート
  • • 事後報告のフォロー

5. 後継者育成プラン

経営者として必要なスキル・知識を身につけるための育成計画を策定します。 段階的に権限を移譲し、社員や取引先の信頼を得られるようサポートします。

  • • 後継者の現状分析（強み・弱み）
  • • 育成計画の策定（3年〜5年計画）
  • • 経営者教育プログラムの紹介
  • • 権限移譲のスケジュール作成

6. M&Aのご相談（後継者がいない場合）

後継者がいない場合、M&A（事業譲渡・売却）も有力な選択肢です。 信頼できるM&A仲介会社と連携し、御社に最適な譲渡先を見つけるサポートをいたします。

  • • M&A仲介会社のご紹介
  • • 企業価値評価（バリュエーション）
  • • 譲渡価格の算定
  • • 譲渡スキームの提案
  • • 税務対策（譲渡所得税の軽減）

解決後のメリット

税負担を大幅に軽減

段階的に株式を移転し、後継者を育成することで、 社員や取引先に不安を与えず、円滑に承継できます。

計画的に承継を進めることで、突然の経営者交代による 混乱を避けることができます。

安心してリタイアできる

引退後の生活資金も確保でき、 「会社のことは後継者に任せて、趣味を楽しもう」と思えるようになります。

実際の承継事例

【卸売業C社：年商5億円、自社株評価額1.2億円】

10年計画で相続税を3,600万円→400万円に

課題

長男に事業を承継したいと考えていたが、自社株の評価額が1.2億円と高く、 相続税が約3,600万円かかると税理士に言われて途方に暮れていた。 後継者である長男も「そんな大金は払えない」と不安を抱えていた。

実施した対策

  • • 10年計画で生前贈与を実施（暦年贈与の活用）
  • • 自社株の評価額を引き下げる対策を実施（役員退職金の支給、不動産購入）
  • • 事業承継税制を活用し、残りの株式は納税猶予を受ける
  • • 後継者育成プランを策定し、段階的に権限を移譲

結果

税負担を約400万円に抑えることに成功（3,600万円→400万円、約3,200万円の節税）。 長男も安心して経営を引き継ぎ、会社は順調に成長。 創業者も安心してリタイアでき、趣味のゴルフを楽しんでいる。

よくあるご質問

Q1. 事業承継対策は何歳から始めるべきですか？

50代から始めるのが理想的です。事業承継は5年〜10年計画で進めるのが一般的ですので、 「まだ早い」ということはありません。むしろ、早く始めるほど選択肢が増え、税負担を軽減できます。 60代、70代でも対策は可能ですが、選択肢が限られてしまうため、できるだけ早めのご相談をおすすめします。

Q2. 生前贈与と相続、どちらが有利ですか？

ケースバイケースです。自社株の評価額、後継者の資金力、会社の状況などによって、 最適な方法は異なります。一般的には、計画的な生前贈与の方が税負担を軽減しやすいですが、 事業承継税制を活用すれば、相続でも納税猶予を受けられます。 御社の状況を分析した上で、最適な方法をご提案いたします。

Q3. 事業承継税制は本当に税金がゼロになるのですか？

要件を満たせば、相続税・贈与税の納税が猶予されます（免除ではありません）。 ただし、後継者が5年間経営を継続し、雇用の8割以上を維持するなど、 厳しい要件があります。要件を満たせなくなった場合、猶予されていた税金を支払う必要があります。 私たちは、要件を満たせるようサポートいたします。

Q4. 後継者がいない場合、どうすればいいですか？

M&A（事業譲渡・売却）も有力な選択肢です。 信頼できるM&A仲介会社をご紹介し、御社に最適な譲渡先を見つけるサポートをいたします。 従業員の雇用を継続してくれる譲渡先を見つけることで、廃業せずに会社を存続させることができます。

Q5. 相談料はかかりますか？

初回相談は無料です。御社の状況をお伺いし、最適なプランをご提案いたします。 ご相談だけで終わっても構いませんので、お気軽にお問い合わせください。
