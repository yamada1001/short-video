大分市在住 / 30代女性 / ライフプランニング

結婚を機に将来のお金について相談しました

結婚を機に将来のお金について不安になり、ファイナンスブレーンさんに相談しました。 ライフプランニングを通じて、今後必要なお金や貯蓄の目標が明確になりました。 保険の見直しも行い、月々の保険料が2万円も削減できたのには驚きました。

担当の方がとても親身になって相談に乗ってくださり、専門用語も分かりやすく説明してくださったので、 初めてでも安心して相談できました。定期的なフォローアップもあり、心強いです。

別府市在住 / 40代男性 / 住宅ローン相談

マイホーム購入時の住宅ローン相談

マイホーム購入を検討していた際、住宅ローンについて相談しました。 複数の金融機関の住宅ローンを比較してくださり、私の収入や家族構成に合った最適なプランを提案していただきました。 変動金利と固定金利のメリット・デメリットも丁寧に説明してくださり、納得して選ぶことができました。

団体信用生命保険についてもアドバイスをいただき、必要な保障が確保できました。 おかげで安心してマイホーム購入を進めることができました。

大分市在住 / 50代男性 / 相続対策

両親の相続対策について相談

両親が高齢になり、相続について考える必要が出てきました。 相続税の仕組みや対策について全く知識がなかったのですが、 ファイナンスブレーンさんに相談したところ、分かりやすく説明していただきました。

生前贈与や生命保険を活用した対策など、具体的なプランを提案していただき、 家族で話し合うことができました。税理士さんもご紹介いただき、 専門家のネットワークの広さにも助けられました。

大分市在住 / 60代女性 / 資産運用

老後資金の運用について相談

退職金の運用について相談しました。銀行の定期預金だけでは利息がほとんどつかないため、 どうしたらよいか悩んでいました。リスクを抑えた資産運用について提案していただき、 投資信託を始めることにしました。

投資は初めてで不安でしたが、リスクとリターンの考え方や、分散投資の重要性など、 基礎から丁寧に教えていただきました。定期的に運用状況を報告してくださるので、 安心して任せることができています。

大分市在住 / 経営者 / 法人向けコンサルティング

会社の財務改善と事業承継対策

中小企業を経営していますが、財務状況の改善と将来の事業承継について相談しました。 キャッシュフローの改善策や、経営者保険の活用など、具体的なアドバイスをいただきました。

事業承継についても、後継者への株式移転の方法や税務対策など、 長期的な視点でプランニングしていただきました。 経営者の立場に立って考えてくださる姿勢がありがたかったです。
