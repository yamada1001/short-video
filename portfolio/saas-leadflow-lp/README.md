# LeadFlow - CRM・営業支援ツール LP

架空のSaaS（CRM・営業支援ツール）のランディングページ

## サービス概要

**LeadFlow**は、営業チームの生産性を最大化するCRM・営業支援ツールです。

### ターゲット
- 中小企業の営業チーム（10-30人規模）
- リモートワークを導入している企業
- シンプルで使いやすいCRMを探している企業

### 主な特徴
- 直感的なUI
- スピード導入（最短3日）
- 売上向上を実感できるレポート機能
- 充実したサポート体制

## デザインコンセプト

**Soft & Friendly**

- パステル系の柔らかい色使い
- メインカラー: 柔らかいパープル (#8B5CF6)
- アクセントカラー: ピンク (#FF6B9D)、ターコイズ (#06B6D4)
- 角丸を多用した親しみやすいデザイン
- グラデーションを効果的に使用

## アニメーション

**リッチ**レベルのアニメーション実装

1. スクロールアニメーション（Intersection Observer）
2. パララックス効果
3. 3Dモックアップ（マウス追従）
4. カードのチルトエフェクト
5. 数字カウンターアニメーション
6. フローティングアニメーション
7. テスティモニアルの自動ハイライト
8. FAQアコーディオン
9. スムーススクロール

## セクション構成

1. **ヘッダー** - 固定ヘッダー、スクロール時にシャドウ追加
2. **ヒーロー** - キャッチコピー、CTA、3Dモックアップ
3. **特徴** - 4つの主要な特徴をカード形式で
4. **主要機能** - 3つの機能を左右交互に配置
5. **数字で見る実績** - 4つの数値指標（カウンターアニメーション）
6. **お客様の声** - 3件のテスティモニアル
7. **料金プラン** - 3つのプラン（スターター/プロフェッショナル/エンタープライズ）
8. **FAQ** - 6つの質問（アコーディオン形式）
9. **最終CTA** - メールアドレス入力フォーム
10. **フッター** - リンク集、コピーライト
