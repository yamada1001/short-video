## 教材形式

### 推奨される教材形式（レッスンタイプ）

各レッスンは以下の形式を組み合わせて構成：

1. **スライド形式（slide）** - 10-15スライド/レッスン
   - 理論説明
   - コンセプト理解
   - ステップバイステップチュートリアル

2. **クイズ形式（quiz）** - 5-10問/レッスン
   - 選択式問題
   - 理解度チェック
   - 実践的な問題

3. **実習形式（editor）** - プロンプト試行
   - 実際にプロンプトを書く
   - サンプルプロンプトのコピー
   - 外部AIツールで実行

4. **課題形式（assignment）** - 実践課題
   - プロジェクトベースの課題
   - 成果物の作成
   - フィードバック投稿

### 1レッスンの標準構成例

**基礎編レッスン**:
- スライド（10枚）：概念説明
- クイズ（5問）：理解度チェック

**実践編レッスン**:
- スライド（8枚）：手順説明
- 実習（3つのプロンプト例）：実際に試す
- クイズ（3問）：応用問題

**応用編レッスン**:
- スライド（5枚）：応用技術説明
- 課題（1つ）：プロジェクト作成
- フィードバック投稿

## 環境構築が必要なAIツール

以下のツールは事前に環境構築が必要：

### 1. Cursor
- **必要なもの**: Windows/Mac/Linux PC
- **インストール**: 公式サイトからダウンロード
- **設定**: APIキー不要（アカウントのみ）

### 2. GitHub Copilot
- **必要なもの**: GitHubアカウント、VS Code等のIDE
- **サブスクリプション**: $10/月
- **インストール**: 拡張機能インストール

### 3. Codeium
- **必要なもの**: 対応IDE（VS Code、JetBrains等）
- **インストール**: 拡張機能インストール
- **設定**: 無料アカウント作成

### 4. Tabnine
- **必要なもの**: 対応IDE
- **インストール**: 拡張機能インストール
- **オプション**: ローカル実行のための環境

### 5. Amazon CodeWhisperer
- **必要なもの**: AWSアカウント
- **インストール**: VS Code拡張機能
- **設定**: AWS認証情報

### 6. Replit AI
- **必要なもの**: Webブラウザのみ
- **インストール**: 不要（Webベース）

### 7. Stable Diffusion
- **必要なもの**:
  - ローカル実行: NVIDIA GPU（VRAM 4GB以上推奨）
  - または Google Colab（無料GPU利用可能）
- **インストール**: AUTOMATIC1111 Web UI
- **設定**: モデルファイルのダウンロード

### 8. Adobe Firefly
- **必要なもの**: Adobeアカウント
- **インストール**: Webブラウザ / Photoshop統合
- **設定**: Creative Cloudアカウント
