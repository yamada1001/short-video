## 目的・KPI

### 主要目標

1. **CPA（顧客獲得単価）の削減**
   - Google広告などの有料広告と連携し、全体的なCPAを最適化
   - Instagram経由での質の高いリード獲得

2. **入社後のギャップ解消**
   - 企業文化や職場環境をリアルに発信
   - 求職者と企業のミスマッチを防ぐコンテンツ作成

3. **ブランディング構築**
   - 「家を売る・買うならハウスドゥ」というブランドイメージの確立
   - 地域密着型で信頼できる不動産会社としてのポジショニング

### 測定指標（KPI）

- フォロワー数の増加率
- エンゲージメント率（いいね、コメント、保存、シェア）
- プロフィールアクセス数
- ウェブサイトクリック数
- DM・問い合わせ件数
- Instagram経由のCPA
- リーチ・インプレッション数

## コンテンツ戦略

### コンテンツの柱（3-4テーマ）

1. **物件紹介・不動産情報**
   - 新着物件の紹介
   - エリア情報・相場解説
   - 住宅購入のポイント

2. **社員紹介・企業文化**
   - スタッフの1日密着
   - 社内イベント・雰囲気
   - 先輩社員のインタビュー
   - 入社後のリアルな声

3. **住まいのお役立ち情報**
   - 住宅ローンの基礎知識
   - リフォーム・リノベーション事例
   - インテリアのアイデア
   - 引っ越しのコツ

4. **ブランドストーリー・実績**
   - ハウスドゥの強み
   - お客様の声・成約事例
   - 地域貢献活動

### トンマナ・デザイン方針

- **色**: ハウスドゥのブランドカラーを基調
- **フォント**: 読みやすさを重視した明朝体・ゴシック体
- **写真**: 明るく、親しみやすい雰囲気
- **テキスト量**: 視覚優先、必要最低限の文字量
