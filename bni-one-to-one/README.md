# BNI ワントゥーワン プロフィールシート

BNI（Business Network International）のワントゥーワン（1to1）ミーティングで使用する自己紹介用のプロフィールシートです。

## 📋 概要

このプロフィールシートは、BNIメンバー同士がより深く知り合うためのツールです。GAINSフレームワーク（Goals, Achievements, Interests, Networks, Skills）を採用し、仕事と個人の両面から自己紹介できるように設計されています。

## 🎯 GAINSフレームワークとは

GAINSは、効果的な自己紹介のための5つの要素の頭文字です：

- **G (Goals)**: ゴール - 仕事/お金/人生の目標
- **A (Achievements)**: 実績 - 仕事/個人的な成果
- **I (Interests)**: 興味 - 仕事/個人的な関心事
- **N (Networks)**: 人脈 - 仕事/個人的なネットワーク
- **S (Skills)**: スキル - 仕事/個人的な能力

## 🚀 使い方

### 1. プロフィール写真の準備

`assets/images/` フォルダに、ご自身のプロフィール写真を `profile.jpg` という名前で配置してください。

- **推奨サイズ**: 正方形（300x300px以上）
- **形式**: JPG または PNG

### 2. 内容の編集

`index.html` を開き、以下の情報をご自身の情報に書き換えてください：

#### 基本情報
- 名前
- 生年月日
- 年齢
- 血液型
- 出身
- 星座
- 家族構成
- 性格
- 勤務先

#### 生い立ち・学歴・職歴
タイムライン形式で記載します。

#### GAINSフレームワーク
各項目（G, A, I, N, S）について、仕事と個人の両面から記入してください。

#### 連絡先
フッター部分の連絡先情報を更新してください。

### 3. プレビュー

ブラウザで `index.html` を開いて、内容を確認してください。

### 4. 印刷・PDF化

ブラウザの印刷機能を使って、PDFとして保存することができます。

- **Chrome**: メニュー > 印刷 > 送信先: PDFに保存
- **Safari**: ファイル > 印刷 > PDFとして保存

## 📱 レスポンシブ対応

以下のデバイスに対応しています：

- **PC** (1000px以上): フルレイアウト
- **タブレット** (768px〜1000px): 中サイズレイアウト
- **スマートフォン** (480px以下): モバイル最適化レイアウト

スマートフォンでは、テーブルが自動的に縦並びになり、読みやすく表示されます。

## 🎨 デザインの特徴

- **BNIブランドカラー**: BNI公式の赤色（#E31E24）を使用
- **プロフェッショナルなデザイン**: ビジネスシーンに適したクリーンなデザイン
- **見やすい表形式**: 情報が整理されて見やすい
- **印刷対応**: 印刷時に最適化されたスタイル

## 💡 カスタマイズのヒント

### カラーの変更

`assets/css/style.css` の以下の部分を編集することで、カラースキームを変更できます：

```css
:root {
    --bni-red: #E31E24;      /* メインカラー */
    --bni-dark: #1a1a1a;     /* ダークカラー */
    --text-primary: #333;     /* テキストカラー */
    --text-secondary: #666;   /* サブテキストカラー */
}
```

### セクションの追加・削除

`index.html` の `<section class="section">...</section>` ブロックを追加・削除することで、セクションをカスタマイズできます。

## 📝 記入のポイント

### Goals（ゴール）
- 具体的な数値目標を入れると良い（例: 3年以内に売上2倍）
- 短期・中期・長期の視点で書く

### Achievements（実績）
- 定量的な成果を記載（例: 年商1億円達成、50社との取引）
- 受賞歴や資格も含める

### Interests（興味）
- 具体的なテーマを記載
- ビジネスチャンスにつながる興味を優先的に

### Networks（人脈）
- 業界や規模を具体的に（例: IT業界100名以上）
- 紹介できる人脈を明確に

### Skills（スキル）
- 具体的なツールや技術名を記載
- レベル感が伝わるように書く

## 🤝 ワントゥーワンでの活用方法

1. **事前共有**: ミーティング前にPDFで共有すると、より深い話ができます
2. **画面共有**: オンラインミーティングでは画面共有で見せながら説明
3. **印刷して持参**: 対面ミーティングでは印刷して渡すと親切です
4. **定期更新**: 3ヶ月〜半年ごとに情報を更新しましょう
