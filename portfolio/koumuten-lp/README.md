# 工務店 LP（ランディングページ）

## 概要
HONEYCOMB.LABOのデザインスタイルをベースにした、工務店向けのランディングページです。

## デザインコンセプト
- **あたたかい家づくり**をテーマにした親しみやすいデザイン
- ハニカム（六角形）モチーフを用いた独自性のあるレイアウト
- 木のぬくもりを感じる自然な配色

## 技術スタック
- **HTML/CSS/JavaScript**: モダンなフロントエンド技術
- **PHP**: サーバーサイド処理（共通パーツのインクルード）
- **レスポンシブデザイン**: モバイルファースト設計（768pxブレークポイント）
- **CSS変数**: デザインシステムの統一管理

## ディレクトリ構造
```
portfolio/koumuten-lp/
├── index.php              # メインページ
├── assets/
│   ├── css/
│   │   ├── common.css     # 共通スタイル（CSS変数、リセット、ユーティリティ）
│   │   └── index.css      # トップページ専用スタイル
│   ├── js/
│   │   └── main.js        # メインJavaScript
│   └── images/            # 画像ファイル
├── includes/
│   ├── header.php         # ヘッダー共通パーツ
│   └── footer.php         # フッター共通パーツ
└── README.md              # このファイル
```
